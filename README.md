# 🗺️ 地水利マップシステム（現場点検・閲覧アプリ）

和歌山市内の消火栓・防火水槽の位置を地図上で確認し、現場から点検結果を直接送信できる専用Webアプリケーションです。iPadやスマートフォンでの利用に最適化されています。

## 🌐 アプリへのアクセス
以下のURLをブラウザ（SafariやChrome）で開いて利用してください。
**[ここにGitHub PagesのURL（https://〜.github.io）を貼り付け]**
※ホーム画面に追加しておくと、全画面のアプリとして快適に利用できます。

## ✨ 主な機能
* **サクサク動くマップ表示:** 7,000件以上の水利データをクラスター化して超軽量で表示。
* **多彩な背景地図:** Googleマップ（標準・航空写真）、国土地理院地図（標準・淡色・航空写真）をワンタップで切り替え。
* **高精度フィルター:** 「公設のみ」「要点検のみ」「特定日以前」など、現場のニーズに合わせた自由な絞り込み検索。
* **ナビゲーション機能:** 移動に合わせて地図が自動でついてくるGPS追従モードを搭載。
* **現場からの直接送信:** 点検モード（※要暗証番号）で入り、現場でフタを開けたその場で点検結果をスプレッドシートへ送信・即時反映。
* **報告書出力:** その日点検した水利だけを抽出し、CSV（Excel用）やPDF報告書として即座に出力可能。

## 🔄 データの更新手順（管理者向け）
現場のアプリ画面は、スプレッドシートのデータと完全連動しています。
定期的なデータ更新は以下の手順で行ってください。

1. 職場のパソコンで最新の水利データ（CSV）を出力する。
2. 本システム連携用の「Googleスプレッドシート」を開き、「生データ」シートにCSVを貼り付ける。
3. 拡張機能「Apps Script」から、データ変換プログラムを実行する。
4. ※過去に現場から入力された点検結果やコメントは、自動で最新台帳へ引き継がれます。

## ⚠️ 注意事項
* **セキュリティ:** 本リポジトリの `index.html` のソースコードは絶対に改変しないでください（システムが停止する恐れがあります）。
* **点検モード:** 現場でのデータ更新（点検モード）に入るには、署内で定めた4桁のパスコードが必要です。
